ZEESEA - EGYPT

Photography, Processed through Adobe Photoshop, 2022

Courtesy of Manh Phi as the photographer

This is a side project that I did with the CYG production team. We want to challenge ourselves by doing a set up production photoshoot with real products.

Process

Set up sketches

We first started with creating the layout sketches for the photoshoot. As the concept artist, it was my duty to come up with the skeches. Since the palette cover is themed around Egyptian Myth, I decided to incorporate the element of sand in the concept.

Behind the scene

Once we all agreed on the final concept sketches, I helped with setting up the products and elements as close to the layouts as possible. After we were sastisfied with the raw photos, I was in charge of cleaning up the first photo and editing the last since it requires more manipulation than the rest.
