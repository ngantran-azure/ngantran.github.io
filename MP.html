ZEESEA - EGYPT

Photography, Processed through Adobe Photoshop, 2022

Courtesy of Manh Phi as the photographer

This is just a side project that I did with the CYG production team. It was both me and the team first time doing a set up production photoshoot with a real product.

Process

Set up sketches

We first started with me creating the layout sketches for the photoshoot. I took me quite a bit of thinking to come up with 4 different views for the Egyptian Myth concept.

Behind the scene

I then helping setting up the products as close to the sketches as possible. I also edited the last photo since it requires more manipulation than the rest. At the end, I'm sastisfied with the set up because we don't haave to manipulate the photos too much.
