Wing Maker

Create a customized wing for your magical creature customer!

Hit 'p' to see what the customer wants

Hit 'd' to display wings

Hit 's' to save the creature

Hit 'c' to go to the next customer!



Description: For this project, the idea was to make a program where it can be interactive. During the process, I was interested in how different one can draw compared to others.

So I made up a scenario where the user is a tailor for fantasy creatures. The "guests" will give out a random prompt, and the tailor would have to design on the egg mannequin the tailored wings.

Example of a out-of-the-box drawing

The user can be creative an not stick to a prompt and use the canvas as a game of 'drawing whatever in the dark'. The goal of this project is to be creative and to have fun!
