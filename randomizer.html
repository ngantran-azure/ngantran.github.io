Random Boba Dispenser



Description:

As a boba enthusiast, I tend to try all the new boba flavour. And until now I have never gotten a bad boba, just got those not exactly up my alley. So I was wondering what if there is a place where you got a random boba drink with random toppings in it. Would you be lucky and get a decent drink? Or your luck is all out and you get a yucky one?

For the "menu", I have a set of flavors and toppings that two thirds is normal and the rest is somthing you wouldn't expect in any drinks (except for healthy ones).

The user can be creative an not stick to a prompt and use the canvas as a game of 'drawing whatever in the dark'. The goal of this project is to be creative and to have fun!
